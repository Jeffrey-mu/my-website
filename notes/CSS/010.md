# 【伪元素小后缀】

> css代码使用scss编写
## 
```scss
.block-card {
    position: relative;
    padding: 0.2rem 1rem;
    background-color: #1b347f;
    color: #fff;
    display: inline-block;
    height: 25px;
    &::after {
        content: '';
        position: absolute;
        top: 100%;
        left: 0;
        margin-top: 0;
        width: 0;
        height: 0;
        border-right: 10px solid #485c98;
        border-top: 0 solid transparent;
        border-bottom: 8px solid transparent;
    }
}
```
