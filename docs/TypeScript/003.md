# 【内置类型 Record】
`Record`的内部定义，接收两个泛型参数；`Record`后面的泛型就是对象键和值的类型


比如我需要一个对象，有 ABC 三个属性，属性的值必须是数字，那么就这么写：

```ts
type keys = 'A' | 'B' | 'C'
const result: Record<keys, number> = {
  A: 1,
  B: 2,
  C: 3
}
```
### 源码声明文件
```ts
/**
 * Construct a type with a set of properties K of type T
 */
type Record<K extends keyof any, T> = {
    [P in K]: T;
};
```
